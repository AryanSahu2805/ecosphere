# EcoSphere — Enterprise Carbon Intelligence Platform

Tracks emissions, sustainability goals, and ESG reporting across
organizations, locations, and departments.

## Tech stack
- Frontend: React, React Router, Axios, Material UI, Recharts
- Backend: Java 21, Spring Boot, Spring Security, Spring Data JPA
- Database: MySQL
- Auth: JWT, role-based authorization
